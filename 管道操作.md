
由于我们将在本章中使用管道操作符 `|>`，以下是对其的简单介绍：

- **`tidyverse`** 使管道操作符 `%>%` 广为人知，它来自 **`magrittr` 包**。  使用管道操作符可以显著简化代码，并使流程更加直观。
- 自 **R 4.1.0** 版本开始，基础 R 也提供了一个管道操作符：`|>`。
    
- **工作原理**：  
    管道操作符 `|>` **会将左侧表达式的输出传递给右侧表达式的第一个参数。**

```
library(magrittr)
x <- c(1, 2, 3, 4, 5)
x %>% sum()
# 输出: [1] 6

1:3 |> sum()
# 输出: [1] 6

c(1, 3, 5, NA) |> mean(na.rm = TRUE)
# 输出: [1] 3

```
`na.rm` 是 R 中许多统计函数（例如 `mean()`、`sum()`、`sd()` 等）常用的一个参数，表示是否 **移除（remove）缺失值 (`NA`)**，从而避免计算出错。
